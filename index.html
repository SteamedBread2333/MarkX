自动完成快捷键

手动触发
Ctrl+E
在任何位置手动触发自动完成菜单。在代码块或引用块内编辑时，底部状态栏会显示提示。
导航选择
↑ ↓
在自动完成菜单中上下移动选择项
确认选择
Enter
确认当前选中的自动完成项并插入
跳转占位符
Tab
跳转到模板中的下一个占位符位置
关闭菜单
Esc
关闭自动完成菜单，取消当前操作

自动完成功能

智能触发
输入 Markdown 关键词或特殊字符（#、*、[、` 等）时自动显示相关模板。也可以输入关键词（如"标题"、"表格"、"代码"）来搜索模板。
手动唤起
按 Ctrl+E（Mac: Cmd+E）可随时手动触发自动完成菜单，查看所有可用模板。
上下文感知
在代码块或引用块内编辑时，自动过滤只显示相关选项。状态栏会显示提示信息。

支持的模板

基础语法
标题、文本格式、链接和图片等基础 Markdown 语法
h1-h6 标题（一级到六级）
bold 加粗文本
italic 斜体文本
code-inline 行内代码
link 链接
image 图片
列表
有序列表、无序列表和任务列表
ul 无序列表（使用 -、* 或 +）
ol 有序列表（数字编号）

nested-ul 嵌套无序列表

表格
各种列数和对齐方式的表格
